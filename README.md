# Clean Water Rush

A charity: water themed mini-game where players collect clean water drops while avoiding polluted ones.

## Game Features

- Tap-based gameplay for mobile and desktop
- 30-second timed rounds with 100-point goal
- Lives system with visual feedback
- Progress tracking toward clean water funding goal
- charity: water brand integration

## How to Play

1. Click "Start Game" to begin
2. Tap blue water drops to collect them (+10 points)
3. Tap yellow jerry cans for bonus points (+25 points)
4. Avoid polluted drops (☠️) or lose points and lives
5. Reach 100 points in 30 seconds to win!

## Technologies

- Vanilla JavaScript
- HTML5
- CSS3

## Setup

Simply open `index.html` in a web browser. No build process required.
